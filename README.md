## Installation

### OSX

`brew install node`

### Ubuntu
`sudo apt update`

`sudo apt install nodejs npm`

### To Run

`npm install`

`npm run dev`

The site should be located at: `localhost:3000`

### To build:

modify `next.config.js` with the path you want to host from, then run

`npm run build`
